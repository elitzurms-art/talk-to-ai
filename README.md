# 🛰️ מכשיר קשר חכם — אפליקציית דפדפן

דבר עם AI בעברית, ישר מהדפדפן. בלי התקנה, בלי שרת, חינם.

**👉 להפעלה:** https://elitzurms-art.github.io/talk-to-ai/

## איך זה עובד
- 🎙️ **זיהוי דיבור** — מובנה בדפדפן (Web Speech API), עברית, חינם.
- 🧠 **מוח** — Google Gemini (`gemini-2.5-flash`), מסלול חינמי.
- 🔊 **הקראה** — מנוע ההקראה של הדפדפן, חינם.

הכל רץ אצלך בדפדפן. מפתח ה-Gemini נשמר רק במכשיר שלך (localStorage) ונשלח רק ל-Google.

## שימוש
1. נכנסים לקישור ב-**Google Chrome** (או Edge).
2. מדביקים מפתח Gemini חינמי — מקבלים כאן: https://aistudio.google.com/app/apikey
3. מאשרים גישה למיקרופון, לוחצים על הכפתור הגדול, מדברים — וה-AI עונה בקול.

## להתקין כאפליקציה
ב-Chrome, בשורת הכתובת יופיע אייקון התקנה (⊕ / "Install"). לחיצה עליו מתקינה את
האפליקציה עם אייקון משלה, ונפתחת בחלון נפרד כמו אפליקציה רגילה.

## תאימות
זיהוי הדיבור עובד ב-Chrome / Edge (מחשב ואנדרואיד). הקראה בעברית תלויה בקול עברי
המותקן במערכת ההפעלה — ברוב המכשירים זה עובד מהקופסה.
